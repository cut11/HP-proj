# ⚡ Magical Harry Potter Universe Web Experience

Welcome to the **Magical Harry Potter Universe**, an immersive and interactive web application built with modern web technologies to bring the wizarding world to life.

## 🌟 Overview

This project is a responsive, single-page React application designed with a deep focus on aesthetics and user experience. It features smooth scroll animations, interactive elements, a custom "magic wand" cursor, and a dark, atmospheric design that perfectly captures the essence of the Harry Potter universe.

## 🛠️ Technologies Used

*   **React 19:** For building dynamic and component-based user interfaces.
*   **Vite:** A blazing fast frontend build tool for an optimized development experience.
*   **Tailwind CSS:** A utility-first CSS framework for rapid UI styling, customized with magical themes (like HP Gold and HP Blue).
*   **Framer Motion:** A production-ready animation library for React, powering the smooth scroll reveals and interactive hover effects.

## ✨ Features & Sections

*   🎵 **Immersive Landing:** An atmospheric entry point with integrated music playback capabilities.
*   🪄 **Interactive Magic Cursor:** A custom cursor that leaves a magical particle trail as you navigate the site.
*   🏰 **Hero Section:** A captivating introduction to the magical world.
*   🦁 **The Four Houses (Home):** Explore Gryffindor, Slytherin, Ravenclaw, and Hufflepuff with dynamic, zooming background cards.
*   🧙‍♂️ **Legendary Characters:** Interactive character cards featuring iconic figures from the series, complete with hover animations and lore.
*   🐉 **Magical Creatures:** Discover fantastic beasts like the Hungarian Horntail, Aragog, Merpeople, and Fawkes in a beautifully animated showcase.
*   🖼️ **Magical Gallery:** A visually stunning gallery featuring a collection of moments and images from the wizarding world.

## 🚀 Getting Started

To run this project locally, follow these steps:

### Prerequisites
Make sure you have Node.js and npm installed on your machine.

### Installation

1.  **Navigate to the project directory:**
    ```bash
    cd "HP proj"
    ```

2.  **Install the dependencies:**
    ```bash
    npm install
    ```

3.  **Start the development server:**
    ```bash
    npm run dev
    ```

4.  **View the site:**
    Open your browser and navigate to the URL provided by Vite (usually `http://localhost:5173`).
